# Cursor Todo App

A simple todo application template built with Vue 3, TypeScript, and Pinia.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Production

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- Vue 3
- TypeScript
- Pinia (State Management)
- Tailwind CSS
- Vite
